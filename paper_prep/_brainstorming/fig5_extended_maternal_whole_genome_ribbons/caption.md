# Draft Caption: Maternal Whole-Genome Recombination Ribbons

**Extended Data Fig. 5 candidate. Maternal whole-genome homologous-competition
ribbon plots replicate the Fig. 5 recombination-compatible signal.** Whole-genome
ribbon views for two maternal transmissions, `PAN027mat_vs_PAN010_joint` and
`PAN028mat_vs_PAN027_joint`, generated by the same homologous-competition method
as Fig. 5. The upper and lower tracks show the two maternal donor haplotypes and
the middle track shows the child query haplotype, with chromosomes concatenated
in chromosome order and scaled by native length. Each query haplotype was split
into 2 kb windows after the raw SweepGA/FastGA f32 many:many PAF was reduced by
the required `sweepga --num-mappings 10:10 --scaffold-jump 0 --scoring ani`
pre-filter. Exact CHM13 centromere windows and windows lacking
interchromosomal support or exceeding the pre-IMPG interchromosomal PAF depth
threshold were removed before `impg similarity`. For each retained 2 kb query
window, the best interchromosomal class winner was compared with the best
same-chromosome homologous class winner. Colored ribbons show only
interchromosomal/non-homologous winners that beat the homologous winner and
belong to exact end-to-end grouped runs of at least 10 kb with mean identity at
least 0.95. Light gray ribbons in the homologous-context render show the
corresponding same-chromosome mother-child homologous chains using the same
exact end-to-end grouping rule. The `PAN027mat_vs_PAN010_joint` render contains
25 drawn high-confidence interchromosomal runs spanning 366 kb, all
acrocentric-acrocentric. The `PAN028mat_vs_PAN027_joint` render contains 104
drawn high-confidence interchromosomal runs spanning 2.07 Mb, dominated by 102
acrocentric-acrocentric runs, with one chr5/chr1 candidate and one
acrocentric-other run.
